# İki kişilik XOX oyunu #
Oyun iki kişi tarafından oynanmaktadır, X yani 1.oyuncu, O ise 2.oyuncudur.
Oyunumuzun temel mantığı, aşağıdan başlamak kaydıyla 1'den 9'a kadar oluşan sayılardan oluşmaktadır.
1.Oyuncu yapıcağı hamleyi 1 ile 9 arasında bir sayı tuşlayarak oyunu başlatıcaktır.
2.Oyuncu ise sırası geldiğinde tekrardan 1.Oyuncunun kullandığı sayıyı kullanmaksızın, 1 ile 9 arasında bir sayı ile oyunu devam ettirecektir.
Oyunculardan birisi seçtiği karakterin 3 tanesini yan yana getirdiği taktirde oyunu kazanıcaktır.
İki oyuncudan herhangi birisi seçtiği karakterin üçünü yanyana getiremezse berabere kalıcaktır.
Ayrıca karakter değişikliği için "Yeniden oynamak istermisiniz?" tarzında bir fonksiyonda bulunmaktadır.


Oyunun dili ve versiyonu :
Python / ver3.9.7

Proje'nin açık kaynak kodlarını yansıtmak için kullanılan program : 
Visual Studio Code.
